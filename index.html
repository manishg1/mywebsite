This website is developed as part of my Cloud Computing course practical project. It demonstrates how websites can be hosted using cloud platforms such as GitHub Pages.

Cloud Computing provides services like storage, hosting, networking, and software over the internet. It helps organizations reduce hardware costs, improve scalability, and access services from anywhere in the world.

In this project, HTML and CSS are used to create a responsive multi-page website, while GitHub cloud hosting is used to deploy and manage the website online.
